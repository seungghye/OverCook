# ⚒ OverCook Modeling

# Info
현재 서비스 중인 OVERCOOKED라는 게임의 프레임워크와 상호작용, 레벨디자인 등을 모작한 멀티플레이 게임을 개발했습니다. 
팀원 : 3명
기획 의도 : 현재 서비스 중인 OVERCOOKED라는 게임을 모작
개발 목표 : 현재 서비스 중인 게임의 디자인을 최대한 유사한 레벨과, UI를 구성해보고자하였습니다.
